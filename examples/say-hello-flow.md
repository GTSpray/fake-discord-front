# Say hello

The `/say hello` command makes the bot greet someone.

## Preview

![Say hello](../tests/snapshots/say-hello-flow.gif)

## Usage

1. Type `/say hello`.
2. Fill in the parameters:
   - **firstname** — first name
   - **secondname** — last name
3. Send the command.

The bot replies in the channel, for example: `hello **Jean Dupont** !`

## Variant

`/say goodbye` says goodbye to someone, using the same kind of parameters.
